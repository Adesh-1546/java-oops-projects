# Java OOP Projects

This repository contains Java projects that demonstrate core Object-Oriented Programming (OOP) concepts through real-world applications.

---

## 🚀 Projects Included

### 1. Employee Payroll System

A simple system to manage employee salary using OOP concepts.

#### Features:

* Add and remove employees
* Calculate salary for:

  * Full-Time Employees
  * Part-Time Employees
* Display employee details

#### Concepts Used:

* Abstraction
* Inheritance
* Polymorphism
* Encapsulation

---

### 2. Car Rental System

A console-based application to manage car rentals with user interaction.

#### Features:

* View available cars
* Rent a car for specific days
* Return rented cars
* Price calculation based on duration
* Customer and rental tracking

#### Concepts Used:

* Encapsulation
* Abstraction
* Collections (ArrayList)
* Real-world system design
